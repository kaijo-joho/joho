カウンターアプリの概要

このワークショップで作成するカウンターアプリは、ブラウザ上で次のように表示され、各ボタンを押すと中央上部の数字が変更されるものです。

ブラウザ上では、左図のように表示されていますが、これをテキストエディタで開くと右図のようなソースで表示されます。このソースに、表示させる要素のコードや実行させたいスクリプトを書くことによって完成させていきます。

カウントアップボタンの作成

ファイルの開き方

デスクトップに移動したHTMLファイルを右クリックし、このアプリケーションで開くを選択します。ブラウザで開くときには、Safari.appをクリックします。また、ソースを編集するときにはテキストエディットをクリックします。

ブラウザで開いても、はじめは何も表示されません。テキストエディットで開いてソースを編集することでブラウザ上にボタン等を表示します。

数字を表示する要素を追加する

まずはじめに、カウンターアプリの数字を表示する要素を追加しましょう。ソースの中にある<!-- 画面に表示する要素 ここから -->と<!-- 画面に表示する要素 ここまで -->のあいだに、次のコードを入力してください。

 0

<と>にはさまれた文字はタグといい、<タグ名 ...>と</タグ名>に囲まれた内容に対して、特定の機能をもたせることができます（タグ自体はブラウザに表示されることはありません）。

今回のdivはコンテンツを入れるための箱を表示するタグです。<div ...>と</div>に囲まれた文字をブラウザ上に表示します。また、id="counter"の部分は、このdivタグにcounterという名前を付けることを意味します。あとでボタンを押したときに、このdivタグの内容を変更するときに指定するために使います。

ここまでできたら、一度保存します（⌘コマンドを押しながらSキーを押すと保存されます）。つづいて、ブラウザでHTMLファイルを開き変更内容が追加されたか確認しましょう。すでにブラウザでHTMLファイルを開いている場合は、再読み込みします（⌘コマンドを押しながらSキーを押すと再読み込みされます）。

カウントアップボタンを追加する

つぎに、カウントアップボタンを追加します。先ほどのdivタグの下に、次のコードを入力してください。

buttonタグはその名の通り、ボタンを表示するためのタグです。<button ...>と</button>に囲まれた文字をボタンの文字とします。onClick="countUp();"の部分は、このボタンをクリックしたときに、countUp()関数を実行することを表します。

カウントアップボタンを押したときの動作

つづいて、カウントアップボタンを押したときの動作をスクリプトに書きます。<!-- スクリプト ここから -->と<!-- スクリプト ここまで -->のあいだに、次のスクリプトを入力してください。

function countUp(){
          let x = getValue();
          counter.textContent = x + 1;
        }

functionは関数といい、特定の機能をもったプログラムのひとかたまりを表します。ここではcountUpという名前の関数を表すという意味です。

xは変数といい、数字や文字などの値を入れておく箱のようなものです。また、letはxが変数であることを宣言するための書き方です。イコール=の右側にあるgetValue()は、いま表示されている数値を取得する関数です。この１行で、変数xを宣言し、そのxの値に、getValue()関数で取得した値を入れることを表しています。

つぎのcounter.textContentは、数字を表示する要素の内容を意味しています。この１行で、数字を表示する要素の内容をイコール=の右側のx + 1、つまりxに１を加えた値に書き換えることを表しています。

ここまでできたら、一度保存します（⌘コマンドを押しながらSキーを押すと保存されます）。つづいて、ブラウザでHTMLファイルを開き変更内容が追加されたか確認しましょう。すでにブラウザでHTMLファイルを開いている場合は、再読み込みします（⌘コマンドを押しながらSキーを押すと再読み込みされます）。カウントアップボタンを押したら、表示されている数字が増えているのが確認できたでしょうか。

カウントダウンボタンを追加するには、カウントアップボタンの前に次のコードを追加します。

リセットボタンを追加するには、カウントアップボタンの後に次のコードを追加します。

カウントダウンボタン・リセットボタンを押したときの動作

スクリプトにカウントダウンボタンとリセットボタンを押したときの動作を書きます。

カウントダウンボタンは、カウントアップボタンと同様に、いま表示されている値をxに入れ、表示内容をx - 1に書き換えます。

function countDown(){
          let x = getValue();
          counter.textContent = x - 1;
        }

リセットボタンは、押したら表示内容を0(ゼロ)にすればよいので、次のように書きます。

function reset(){
          counter.textContent = 0;
        }

ここまでできたら、一度保存します（⌘コマンドを押しながらSキーを押すと保存されます）。つづいて、ブラウザでHTMLファイルを開き変更内容が追加されたか確認しましょう。すでにブラウザでHTMLファイルを開いている場合は、再読み込みします（⌘コマンドを押しながらSキーを押すと再読み込みされます）。カウントダウンボタンとリセットボタンはちゃんと動いたでしょうか。

カウントダウンボタンの下限を決める

ここまでの状態では、カウントダウンボタンを押していくと、0(ゼロ)より小さいマイナスの数が表示されます。表示される数字の下限を0(ゼロ)と決めることで、マイナスの数を表示しないようにします。

例えば、いま表示されている数字が１のとき、カウントダウンボタンを押せば、表示は０になります。では、いま表示されている数字が０のとき、カウントダウンボタンを押すと、０より１小さい数字である−１(マイナス１)が表示されてしまいます。つまり、いまの表示が０より大きい場合はいまの表示から１引いた数を表示して、０の場合（または０より小さい場合）は０を表示するようにすれば、０より小さい数が表示されることはありません。

このように、いま表示されている数によって処理を分ければよいのです。これを条件分岐といい、スクリプトには次のように書きます。

if(条件){
          条件が合っている場合の処理;
        }else{
          それ以外の場合の処理;
        }

今回は、getValue()関数で受け取った値をxとしていますが、xの値が0より大きい(x > 0)場合とそれ以外の場合で条件分岐をします。

if(x > 0){
          counter.textContent = x - 1;
        }else{
          counter.textContent = 0;
        }

これをcountDown()関数に組み込むと、次のようになります。

function countDown(){
          let x = getValue();
          if(x > 0){
            counter.textContent = x - 1;
          }else{
            counter.textContent = 0;
          }
        }
